# REACT Expo

React Native News application country based South Africa

## Run Locally

Clone the project

```bash
  git clone https://github.com/ShaneKolkoto/News-App.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the App

```bash
  expo start
```


## Config Folder Variables
 - config.js 
    - `API_KEY`
    - `endpoint`
    - `category`
    - `country`
    - `pageSize`
